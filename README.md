# AI-Internship-finder-agent
A autonomus ai agent which will find internship according to the resume and a mock interview simulator which will help out to find the user real time accuracy and ability.



## 🚀 Running the Project

Follow the steps below to start the AI project locally:

#1 Activate the Virtual Environment

[.venv\Scripts\activate]

#2 Start Flask server 

[python app.py]
